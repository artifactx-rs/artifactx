# Code of Conduct

ArtifactX is a technical project for package repository operators and maintainers. Keep discussions useful, specific, and respectful.

## Expected behavior

- Focus criticism on code, docs, tradeoffs, and evidence.
- Assume maintainers and contributors are trying to make the project better.
- Provide reproduction steps, logs, and constraints when reporting problems.
- Respect private security reports and embargoed details.

## Unacceptable behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private keys, tokens, private package URLs, or confidential vulnerability details.
- Repeated low-signal demands that ignore maintainer decisions or the current feature freeze.

Maintainers may edit, hide, lock, or remove issues, discussions, comments, and pull requests that violate these rules.
